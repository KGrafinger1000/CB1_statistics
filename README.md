# CB1_statistics

Statistical analysis workflows for CB1 receptor affinity (Ki) and potency (EC50) datasets.

## Files

- `CB1_Ki_LMM.ipynb`
  Linear mixed-effects modeling workflow for CB1 affinity data.

- `CB1_EC50_LMM.ipynb`
  Linear mixed-effects modeling workflow for CB1 activation data.

- `replicates_Ki_checked_for_statitic.xlsx`
  Processed affinity dataset used for statistical analysis.

- `subdatset_EC50_df_checked.xlsx`
  Processed potency dataset used for statistical analysis.

## Software

Analyses were performed in Python using Jupyter Notebook.

Required packages are listed within the notebooks.
